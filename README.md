# Python Mini Projects

This repository contains a collection of small Python scripts created for learning and practicing different programming concepts. The projects cover basic algorithms, simple automation, and computer vision experiments.

## Projects Included

### 1. alphabets_keys_test_drive.py
A simple program that tests keyboard alphabet inputs and demonstrates basic input handling in Python.

### 2. armstrong.py
A script that checks whether a number is an Armstrong number. This project demonstrates mathematical logic and conditional statements.

### 3. check_weather.py
A small script that retrieves weather information using an API and displays it in the terminal.

### 4. checky_pro.py
A simple utility script created for experimenting with Python functions and user input.

### 5. detect_face.py
A computer vision script that detects faces using a webcam and OpenCV.

### 6. ecapture.py
A script that captures images from the computer camera using Python.

### 7. game.py
A small Python game created to practice loops, conditions, and user interaction.

### 8. vowe.py
A script that counts vowels in a word or sentence entered by the user.

## Technologies Used

- Python
- OpenCV
- APIs
- Basic Algorithms

## How to Run the Scripts

1. Install Python
2. Clone this repository
3. Run any script using the terminal:

```bash
python filename.py
```

Example:

```bash
python armstrong.py
```

The goal of this repository is to practice Python programming and explore different areas such as automation, algorithms, and basic computer vision.
